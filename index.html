⚙ Google Sheets Sync Setup

Life OS syncs via JSONBin.io — a free cloud JSON store. Setup takes 2 minutes. The Master Key is needed to write data (reads are public).

Setup steps:
1. Go to jsonbin.io → Sign up free
2. Click Create Bin → paste {"init":true} → Save
3. Copy the Bin ID from the URL (long string after /b/)
4. Go to Account → API Keys → copy your Master Key (starts with $2a$10$)
5. Paste both below → Test → Save
